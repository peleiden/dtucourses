Motivation

Denne hjemmeside er inspireret af den fantastiske DTU Course Analyzer. Vi følte dog, at den kunne bruge nogle tilføjelser som fx et samlet mål for kursers kvalitet, og vi ville også gerne prøve at se på nogle statistikker på tværs af kurser, hvilket ikke er muligt med DTU Course Analyzer. Derudover er DTU ikke blevet opdateret i noget tid og mangler derfor nogle kurser samt de nyeste karakterer og evalueringer.

Teknologier

Alle kursusdata blev scrapet fra DTU Kursusbasen med Python 3.7. Python blev også brugt til at beregne sammensatte scorer og statistikker på tværs af kurser. Hjemmesiden er lavet i Angular 8 og hostet på GitHub Pages. Styling er baseret på Det Fælles Designsystem, og ikoner er taget fra Angular Material. Den fulde kildekode er tilgængelig på GitHub.

Uddybende forklaring af scorer

Alle karaktergennemsnit er taget fra den senest tilgængelige eksamen. Det samme gælder evalueringer, der ligger til grund for de resterende kernetal givet til kurser. Disse er standardiseret, så de fordeler sig mellem 0 og 4, hvilket også gør sig gældende for karaktergennemsnittene til beregning af sammensatte scorer, så den er sammenlignelig.

Ølscoren er givet ved karaktergennemsnittet minus arbejdsbyrde. Kvalitet er ølscore plus indlæring plus samlet indtryk. Disse blev efter beregning igen standardiseret, så de fordeler sig mellem 0 og 4.
